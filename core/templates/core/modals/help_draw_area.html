{% translate 'How to draw an area?' %}
  1. {% translate "Zoom on the zone where you will draw by using your mouse wheel or touchpad or fingers or the zoom-in / zoom-out buttons on the map" %}
  2. {% translate "Then draw the border of the area by clicking on the map" %}
    • {% translate "first click will initiate the border by building a first point" %}
    • {% translate "second click will draw a the first side of the border" %}
    • {% translate "third click will draw a the second side of the border" %}
    • {% translate "and so on ..." %}
    • {% translate "close the border by clicking back on the first point or by double clicking when you draw the last side." %}
  3. {% translate "If you want to draw again your area:" %}
    • {% translate "close it by double clicking on the last point (or by clicking on the first point)" %}
    • {% translate "click on the delete button on the map" %}

{% translate "Tip: click on 'Shift' key while you draw to activate freehand drawing." %}
