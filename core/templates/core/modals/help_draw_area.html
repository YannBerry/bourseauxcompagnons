{% trans 'How to draw an area?' %}
  1. {% trans "Zoom on the zone where you will draw by using your mouse wheel or touchpad or fingers or the zoom-in / zoom-out buttons on the map" %}
  2. {% trans "Then draw the border of the area by clicking on the map" %}
    • {% trans "first click will initiate the border by building a first point" %}
    • {% trans "second click will draw a the first side of the border" %}
    • {% trans "third click will draw a the second side of the border" %}
    • {% trans "and so on ..." %}
    • {% trans "close the border by clicking back on the first point or by double clicking when you draw the last side." %}
  3. {% trans "If you want to draw again your area:" %}
    • {% trans "close it by double clicking on the last point (or by clicking on the first point)" %}
    • {% trans "click on the delete button on the map" %}

{% trans "Tip: click on 'Shift' key while you draw to activate freehand drawing." %}
